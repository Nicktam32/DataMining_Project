Project Overview

Summary

The project aims to develop accurate forecasting models for Hong Kong property prices using historical data from the Rating and Valuation Department on rental statistics from 1999-2023 as well as GDP, population, and housing stock/vacancy data. This is motivated by rising housing costs creating challenges for buyers. A decision tree model explaining 72% of variance in Class A rents and an LSTM model capable of capturing time series patterns like seasonality were selected. Time series visualizations identified August tending higher and April lower than trends. To deliver these insights, a website (https://math-3836-data-mining-project.vercel.app/) is proposed to integrate data sources, improve visualizations, collaborate with real estate agents, offer mortgage calculations, community features, and leverage influencer marketing. The overall focus is on data-driven decision making for investors, the public, and policymakers in the Hong Kong property market.

Objectives

Objective 1: To utilize data visualization tools (SVG, amCharts)

Objective 2: To apply JavaScript to manipulate data

Objective 3: To discover insights into the Hong Kong property market

Objective 4: To design an interactive and user-friendly interface

Objective 5: To develop an accurate and reliable forecasting model for property prices

Objective 6: To able data-driven decision-making for investors, general people, and policymakers

Methodology
  1. Import necessary CSS and JavaScript libraries, such as Bootstrap, amCharts, and D3.js

  2. Develop a page structure consisting of a navigation bar and a content section using Bootstrap's grid system

  3. Visualize the data by using the amCharts library and D3.js library

  4. Apply data mining technique into a web product
