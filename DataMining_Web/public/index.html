Project Overview

Summary

Introduction & Background:
- The project aims to develop an accurate forecasting model for property prices in Hong Kong to enable data-driven decision making for investors, general public, and policymakers.
- The motivation is the rising housing costs which create challenges for buyers in terms of high prices, down payments, and transaction costs.
- The data is sourced from the Rating and Valuation Department and includes property rental statistics from 1999-2023 as well as GDP, population, and housing stock/vacancy data.

Model Selection:

- A decision tree model was built which could explain around 72% of the variance in the target variable (Class A Hong Kong rents).
- An LSTM (long short-term memory) model was also selected due to its ability to model time series data, adapt to new data, capture long-term dependencies/patterns like seasonality.

Model Visualization:

- Visualizations like time series plots, seasonal decomposition, and ARIMA forecasts were used to analyze the data patterns.
- The seasonal plot identified August as tending higher than the trend and April tending lower.

Product Selection:
- A website (https://math-3836-data-mining-project.vercel.app/) is proposed to integrate data sources, improve visualizations, collaborate with real estate agents, offer mortgage calculations, community features, and leverage influencer marketing.

The overall focus is on developing property price forecasting models from historical data to provide insights to stakeholders, with plans to deliver this via an interactive website.

Objectives

Objective 1: To utilize data visualization tools (SVG, amCharts)

Objective 2: To apply JavaScript to manipulate data

Objective 3: To discover insights into the Hong Kong property market

Objective 4: To design an interactive and user-friendly interface

Methodology
  1. Import necessary CSS and JavaScript libraries, such as Bootstrap, amCharts, and D3.js

  2. Develop a page structure consisting of a navigation bar and a content section using Bootstrap's grid system

  3. Visualize the data by using the amCharts library and D3.js library
